# My App

React와 React Router를 사용한 웹 애플리케이션입니다.

## 1. 환경 세팅

**필요 환경:**
- Node.js 버전 18.0.0 이상
- npm (Node.js와 함께 설치됨)

**설치 확인:**
```bash
node --version
npm --version
```

## 2. 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start
```

## 3. 확인 방법

브라우저에서 `http://localhost:3000` 접속

- 홈페이지: `http://localhost:3000/`
- Hello 페이지: `http://localhost:3000/hello`
